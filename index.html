RecCall

Record and call context shortcuts in Cursor IDE with MCP. A simple Model Context Protocol server that lets you record context/instructions once and call them instantly in your AI conversations.

üöÄ Available Commands

rec Record a new shortcut with context/instructions
rec_list List all stored shortcuts
rec_update Update/replace an existing shortcut
rec_delete Delete a shortcut (idempotent operation)
rec_purge Purge all shortcuts (with confirmation)
call Call a stored shortcut

üí° Example Usage

Once configured in Cursor, you can use these commands in your AI chat:

List shortcuts "List all my shortcuts"
Call shortcut "Call the 'react-component' shortcut"
