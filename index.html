‚ö° Quick Start

1. Record a Shortcut

Save a complex instruction as a shortcut:

rec "deploy" "build the project, run tests, and deploy to production"

2. Recall the Shortcut

Use it in Cursor chat:

call deploy

3. Manage Shortcuts

List all your shortcuts:

rec_list

üõ†Ô∏è Available Commands

rec "name" "instruction" - Record a new shortcut

call name - Recall a shortcut

rec_list - List all shortcuts

rec_update "name" "new instruction" - Update a shortcut

rec_delete "name" - Delete a shortcut

rec_purge - Delete all shortcuts
