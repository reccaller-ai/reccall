🚀 Available Commands

rec Record a new shortcut with context/instructions
rec_list List all stored shortcuts (equivalent to rec -l)
rec_update Update/replace an existing shortcut (equivalent to rec -u)
rec_delete Delete a shortcut (idempotent operation)
rec_purge Purge all shortcuts (with confirmation)
call Call a stored shortcut

📦 Installation

Prerequisites

  • Node.js 18 or higher
  • Cursor IDE with MCP support
  • npm or yarn

Quick Setup

# 1. Clone the repository git clone https://github.com/reccaller-ai/reccall.git cd reccall # 2. Install dependencies npm install # 3. Build the project npm run build # 4. Note the absolute path pwd # Copy this path - you'll need it for Cursor config

Configure Cursor IDE

Add this to your Cursor configuration file:

{ "mcpServers": { "reccall": { "command": "node", "args": ["/FULL/PATH/TO/reccall/index.js"] } } }

Configuration file locations:

  • macOS: ~/.cursor/config.json
  • Linux: ~/.cursor/config.json
  • Windows: %APPDATA%\Cursor\User\globalStorage\saoudrizwan.claude-dev\settings\cline_mcp_settings.json

⚠️ Important: After adding the configuration, completely restart Cursor for the changes to take effect.

💡 Actual Usage Examples

Once configured, you interact with RecCall through natural language in Cursor's AI chat:

Recording Shortcuts

# Natural language approach Save a shortcut called "react-component" with the context: "Create React functional components with TypeScript. Use proper prop types, add JSDoc comments, implement error boundaries, and use React.memo for performance optimization." # Direct tool usage Use the rec tool to save this: Shortcut: apitest Context: When testing APIs, always include error handling, timeout configuration, and proper request/response logging. Use axios with interceptors for auth tokens.

Listing Shortcuts

# Natural language List all my shortcuts # Direct tool usage Use the rec_list tool to show me what shortcuts I have

Updating Shortcuts

# Natural language Update my 'apitest' shortcut with: "When testing APIs, always include comprehensive error handling, timeout configuration, request/response logging, authentication testing, and rate limiting validation. Use axios with interceptors for auth tokens and implement retry logic." # Direct tool usage Use the rec_update tool to change my 'react-component' shortcut to: "Create React functional components with TypeScript, proper prop types, JSDoc comments, error boundaries, React.memo for performance, and include unit tests with React Testing Library."

Deleting Shortcuts

# Natural language Delete my 'old-shortcut' shortcut # Direct tool usage Use the rec_delete tool to remove the 'temp-shortcut' shortcut

Calling Shortcuts

# Natural language Call the 'apitest' shortcut # Direct tool usage Use the call tool to get my 'react-component' guidelines

Purging All Shortcuts

# Natural language Purge all my shortcuts # Direct tool usage Use the rec_purge tool to delete all stored shortcuts

Note: The AI will automatically call the appropriate MCP tools based on your natural language requests. You don't need to use the tool names directly.
