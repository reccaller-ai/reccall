‚ö° Quick Start

1. Record a Shortcut

Save a complex instruction as a shortcut:

# Cursor IDE (MCP) rec "deploy" "build the project, run tests, and deploy to production" # VSCode (Extension) Ctrl+Shift+R ‚Üí Enter shortcut name and context # Warp Terminal / CLI reccall rec deploy "build the project, run tests, and deploy to production"

2. Recall the Shortcut

Use it across all platforms:

# Cursor IDE (MCP) call deploy # VSCode (Extension) Ctrl+Shift+C ‚Üí Select "deploy" # Warp Terminal / CLI reccall call deploy

3. Manage Shortcuts

List all your shortcuts:

# Cursor IDE (MCP) rec_list # VSCode (Extension) Ctrl+Shift+L # Warp Terminal / CLI reccall list

üõ†Ô∏è Available Commands

Cursor IDE (MCP)

reccall call name - Recall a shortcut

reccall list - List all shortcuts

reccall search query - Search shortcuts

reccall update name context - Update a shortcut

reccall delete name - Delete a shortcut

reccall purge --yes - Delete all shortcuts

reccall import/export file - Backup and restore

üîß Manual Setup

If you prefer manual installation:

1. Install Globally

npm install -g reccall

2. Configure Each Platform

Cursor IDE (MCP)

Add to your Cursor MCP configuration:

{ "mcpServers": { "reccall": { "command": "reccall-mcp", "args": [] } } }

VSCode Extension

code --install-extension reccall

Warp Terminal

echo 'source $(which reccall-warp.sh)' >> ~/.zshrc

CLI Tool

Available globally as reccall command after installation.
